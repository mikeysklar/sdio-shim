# sdio-shim

A direct-connect shim PCB that mates the Adafruit SDIO breakout to a Feather — no breadboard required.

## About

This board is a solder-free, direct-connect shim compatible with the [Adafruit SDIO breakout](https://www.adafruit.com/), letting you connect it straight to a Feather. It replaces the usual breadboard-and-jumper-wire hookup with a single rigid board.

Design files are in [`kicad/`](kicad/).
